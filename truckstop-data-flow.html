Security Controls
Row Level Security (RLS)Every table enforces RLS via owner_id. Each carrier only accesses their own data.
AES-256-GCM EncryptionPartner OAuth tokens encrypted at rest with unique IV before database storage.
JWT AuthenticationEvery API request requires a valid Supabase JWT. Backend validates before processing.
US-Only InfrastructureAll servers (Vercel, Supabase, Anthropic) are US-based. No cross-border transfer.
Token IsolationPartner OAuth tokens never leave the backend. Only used server-side for authorized API calls.
Audit LoggingEvery data mutation is logged with timestamps, old/new values, and user ID for full traceability.
